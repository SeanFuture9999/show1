# 重生抽卡復仇記 — Phase 0 定錨文件

## 願景聲明

> 前世被兄弟和妻子推下樓的商界精英，重生後獲得神秘懷錶的抽卡之力。
> 用抽到的能力、人脈和情報，一步步撕開所有人的面具，展開最爽的復仇。

## 遊戲類型

文字劇情 + 抽卡 + 表演式互動（類爽劇互動小說）

## 美術風格

- **主風格**：暗黑像素風（16-bit 精細像素）
- **色調**：賽博龐克暗色系（深藍、紫、霓虹粉）
- **氛圍**：雨夜、霓虹、電影感打光
- **參考**：PixelLab Round 4 測試圖（style1_classic_rpg 為基準）

## 目標玩家

| 項目 | 設定 |
|------|------|
| 年齡層 | 25-40（看爽劇的主力族群） |
| 遊戲經驗 | 輕度～中度（不需要操作技巧） |
| 遊玩情境 | 通勤碎片 / 睡前追劇 / 排隊等待 |
| 單次遊玩 | 5-15 分鐘 |
| 付費意願 | 免費為主，願意看廣告換資源 |
| 典型玩家 | 愛看大陸爽劇/重生復仇網路小說的手機用戶 |

## 核心體驗

**爽感**（第一優先）
- 打臉仇人的爽快感
- 抽到好卡的驚喜感
- 看主角逆襲的代入感

**沉浸感**（第二優先）
- 打字機效果的文字帶入
- 配合情節的光效特效
- 關鍵場景的像素插畫 CG

## 商業模式

```
基礎免費 + 廣告 + 付費加速

免費玩家：
├── 完整主線劇情（不鎖章節）
├── 每日免費抽卡次數
├── 看廣告 → 額外抽卡 / 加速劇情推進
└── 體力系統（自然回復）

付費玩家：
├── 去廣告
├── 額外抽卡石（更多抽卡機會）
├── 月卡（每日領抽卡石 + 體力）
├── 隱藏劇情線解鎖（後宮線/暗線/IF 線）
└── 限定卡池（付費限定 SSR）
```

## 世界觀設定

### 背景
現代都市商戰世界。主角「林昊」原是知名企業聯合創辦人，被最信任的合夥人兼兄弟「陳衛」與妻子「蘇婉」聯手背叛，在公司頂樓被推下。

### 重生
死亡瞬間，時間倒退。主角醒來發現回到了三年前 — 一切陰謀尚未開始。手中多了一只來歷不明的古董懷錶。

### 懷錶系統
打開懷錶蓋 = 進入抽卡介面。懷錶會在關鍵時刻發光震動，提示可以抽卡。抽出的卡牌在現實中生效：
- **能力卡**：暫時獲得超凡能力（讀心、預知、超強記憶等）
- **人脈卡**：召喚/遇見關鍵人物（律師、駭客、記者等）
- **情報卡**：獲得關鍵情報（商業機密、黑歷史、隱藏關係）
- **資源卡**：獲得資源（資金、證據、關鍵物品）

### 限制
- 懷錶能量有限，不能無限抽（= 體力/付費設計）
- 抽到的卡有使用期限或次數限制
- 部分 SSR 卡有副作用（增加戲劇張力）

| 稀有度 | 顏色 | 機率 | 代表卡牌範例 |
|--------|------|------|-------------|

| SR 史詩 | 🟣 紫 | 12% | 預知未來 1 小時、頂級駭客、銀行內線 |

## 章節結構

```
序章   — 墜樓・重生・懷錶覺醒（教學 + 開場）

Ch.1  「螻蟻」— 公司小主管（教學關）
       → 學會抽卡基礎，第一次打臉，建立自信

Ch.2  「背刺」— 背叛的秘書/助理
       → 發現內鬼，揪出出賣情報的人，情報戰入門

Ch.3  「鯊魚」— 對手公司 CEO
       → 正面商戰對決，收購攻防，大場面

Ch.4  「蛇蠍」— 背叛的妻子
       → 情感線高潮，最虐心也最解氣的一章

Ch.5  「兄弟」— 最信任的合夥人
       → 真相揭露，正面對決推你下樓的人

Final 「棋手」— 幕後操控一切的黑手
       → 原來一切都是更大的棋局，最終決戰
```

## 呈現方式

### 文字演出
- 打字機效果逐字顯示（可調速度/一鍵顯示全文）
- 角色對話有頭像框
- 旁白/內心戲用不同字體或顏色區分
- 關鍵台詞放大 + 震動效果

### 插畫 CG
- 每章 3-5 張關鍵場景像素插畫（PixelLab 生成）
- 全屏展示，搭配音效
- 解鎖後可在「回憶」中重看

### 光效特效
- 懷錶發光（抽卡前兆）
- 角色怒氣/殺氣（紅色粒子）
- 打臉瞬間閃白 + 震動
- 場景氣氛光（雨、霓虹、火焰）

### 表演式戰鬥
- **簡單版**：像素角色抖動 + 大字「啪！」「碰！」+ 音效 + 螢幕震動
- 參考「叫我官老爺」的用刑表演
- 不需要玩家操作，純演出
