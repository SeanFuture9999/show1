# 重生抽卡復仇記 — Phase 1 深度提問摘要

## 核心循環

### 秒級（最小操作）
- 點擊螢幕推進文字（打字機效果）
- 選擇「忍」或「用卡」

### 分鐘級（一個完整小循環）
```
劇情鋪陳（1-2 分鐘）
  → 壞人使壞，囂張值上升
  → 主角隱忍，累積抽卡值
  → 抽卡值滿，懷錶震動
  → 抽卡！
  → 選擇：用？還是繼續忍？
    → 用 → 囂張值下降 → 打臉演出（爽！）
    → 忍 → 壞人更囂張 → 被欺負劇情（虐！但之後更爽）
```

### 小時級（一次遊玩 10-15 分鐘）
- 推進 1 章的 1/3 ~ 1/2 劇情
- 經歷 1-2 次「忍 vs 用」抉擇
- 看到 1-2 個打臉場景或被虐場景

### 跨天級（長期目標）
- 推進主線章節（序章→Ch.1→Ch.2...）
- 收集稀有卡牌
- 解鎖隱藏劇情線（忍到底才能看）
- 全 CG 插畫收集

### 核心爽點
**「忍到極限再一次爆發」的瞬間** — 累積越多，打臉越狠，隱藏劇情越精彩

## 體驗支柱（按優先級）

| 優先級 | 支柱 | 說明 |
|--------|------|------|
| ⭐⭐⭐ | 敘事爽感 | 爽劇節奏的打臉復仇故事 |
| ⭐⭐⭐ | 策略抉擇 | 忍 vs 用的 push your luck 緊張感 |
| ⭐⭐ | 收集 | 卡牌收集 + CG 圖鑑 |
| ⭐ | 表現 | 不同打臉方式的演出差異 |

## 核心機制詳細設計

### 囂張值系統
```
壞人囂張值 [████████░░] UI 可見進度條

- 壞人使壞事件 → 囂張值 +N
- 玩家打卡打臉 → 囂張值 -N（依卡片等級）
- 累積多張一起用 → 更高傷害 + 隱藏劇情解鎖
- 囂張值到 100% → 壞人勝出 → Game Over
- Game Over → 看廣告 → 重生到記錄點
```

### 抽卡值累積
```
壞人使壞 → 主角隱忍 → 抽卡值累積
抽卡值滿 → 懷錶發光 → 可以抽卡
抽到卡 → 選擇用或繼續忍
```

### 忍 vs 用的風險回報
| 選擇 | 風險 | 回報 |
|------|------|------|
| 馬上用 | 低（安全降低囂張值）| 小打臉，基本劇情 |
| 忍 1 次 | 中（囂張值繼續上升）| 中等打臉 + 額外對話 |
| 忍到極限 | 高（接近 Game Over）| 最大傷害 + 隱藏劇情解鎖 |

### 存檔與回溯
- 章節內不可讀檔，選了就是選了
- 可看廣告回溯到前一個記錄點
- 章節之間可重玩已完成的章節

## 廣告觸發點（共 4 個）

| 觸發點 | 情境 | 玩家心理 |
|--------|------|----------|
| Game Over 重生 | 囂張值爆表 | 「不甘心！再來一次」 |
| 回溯記錄點 | 選錯想反悔 | 「早知道就先打臉」 |
| 加速累積抽卡值 | 忍不住了想快抽 | 「快給我卡！」 |
| 解鎖隱藏事件線索 | 好奇隱藏內容 | 「一定有什麼秘密」 |
| 每日免費禮包 | 每天登入 | 日常習慣 |

## 卡池系統

### 卡片類型（混合型）
| 類型 | 說明 | 範例 |
|------|------|------|
| 能力卡 | 暫時超能力 | 讀心 30 秒、超強記憶、預知 |
| 人脈卡 | 召喚關鍵人物 | 律師、駭客、記者、警察 |
| 情報卡 | 獲得秘密 | 商業機密、黑歷史、帳本 |
| 資源卡 | 獲得資源 | 資金、證據、關鍵物品 |

### 稀有度
| 等級 | 顏色 | 機率 | 囂張值傷害 |
|------|------|------|-----------|
| N | ⚪ 白 | 60% | -5~10 |
| R | 🔵 藍 | 25% | -15~25 |
| SR | 🟣 紫 | 12% | -30~50 |
| SSR | 🟡 金 | 3% | -60~80 + 隱藏劇情 |

## 章節結構

### 節奏設計（以 Ch.1 為例）
```
Ch.1「螻蟻」— 小主管

段落 1: 鋪陳 — 小主管欺壓新人（囂張值 +15）
段落 2: 使壞 — 搶功勞、甩鍋（囂張值 +20）→ 可抽卡①
段落 3: 升級 — 拉攏其他同事孤立主角（囂張值 +20）→ 可抽卡②
段落 4: 高潮 — 在大會議上公開羞辱（囂張值 +25）→ 可抽卡③
段落 5: 打臉 — 依據累積的卡片，反擊場景
段落 6: 結算 — 小主管下場 + 下章預告

前期每章 3-4 次抽卡，後期更多
囂張值上限 100，玩家必須在 Game Over 前打臉
```

## 內容量估算（MVP）

| 內容 | 數量 | 工具 |
|------|------|------|
| 劇情文字 | ~50,000 字（3 章） | Claude 生成 + 人工審稿 |

## 關鍵風險

| 風險 | 等級 | 對策 |
|------|------|------|
| 劇情品質不夠「爽」 | 🔴 高 | 參考熱門爽劇節奏，找人試讀 |
| 囂張值平衡難調 | 🟡 中 | 先做 Ch.1 反覆測試手感 |
| PixelLab 風格不統一 | 🟡 中 | 統一 prompt 前綴（已驗證可行） |
| 一個人做太慢失去動力 | 🟡 中 | MVP 只做 3 章，快速看到成果 |
| 廣告太多玩家反感 | 🟡 中 | 控制在 4 個觸發點，都是玩家主動選擇 |
