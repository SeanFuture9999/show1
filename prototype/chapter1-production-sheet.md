## 🔀 出牌分支流程圖

### 出牌點 #1：搶功勞（Scene 01）
```
手牌：❶愚者 / ❷暴食 / ❸審判

❶ 愚者 ⭐最佳 → 裝傻鞠躬，朴在成放下警戒
❷ 暴食 ❌錯誤 → 沒證據出牌失敗，囂張值+5
❸ 審判 ✅可用 → 小打臉，但朴在成開始警戒
         ↓
    全部匯合 → 旻赫記帳本「搶功勞，第一次」
```

### 出牌點 #2：深夜加班（Scene 03）
```
手牌：❶愚者 / ❷暴食 / ❸審判 / ❹已讀

❶ 愚者 ❌錯誤 → 空房間裝傻沒意義，囂張值+5
❷ 暴食 ⭐最佳 → 查帳本！14筆4700萬假帳 ← 關鍵證據
❸ 審判 ✅可用 → 找到部分痕跡，冰山一角
❹ 已讀 ❌錯誤 → 場合不對，囂張值+5
         ↓
    全部匯合 → 冰美式字條「喂快點下班啦」
```

### 出牌點 #3：會食逼酒（Scene 04）
```
手牌：❶愚者 / ❷暴食 / ❸審判 / ❹已讀

❶ 愚者 ⭐最佳 → 喝酒忍辱，偷拍法人卡 → 觸發智媛爆發名場面
❷ 暴食 ✅可用 → 問「用法人卡？」小刺，朴在成警覺
❸ 審判 ❌錯誤 → 酒桌翻證據？讀空氣！
❹ 已讀 ❌錯誤 → 配燒酒？場合不對
         ↓
    全部匯合 → Scene 05：朴在成甩鍋報復
```

### 出牌點 #4：茶水間威脅（Scene 07）
```
手牌：❶愚者 / ❷暴食 / ❸審判 / ❹已讀 / ❺該隱

❶ 愚者 ⭐最佳 → 低頭走開，去停車場拍最後證據
❷ 暴食 ❌錯誤 → 走廊翻帳本？沒觀眾！
❸ 審判 ❌錯誤 → 走廊舉證？需要舞台！
❹ 已讀 ✅可用 → 小反擊，但朴在成開始銷毀證據
```
