# 重生抽卡復仇記 — 遊戲設計文件 v0.5

---

## 1. 概覽

### 1.1 願景聲明

> 前世被兄弟和妻子推下樓的商界精英，重生後獲得神秘懷錶的抽卡之力。
> 用抽到的能力、人脈和情報，一步步撕開所有人的面具，展開最爽的復仇。

### 1.2 設計支柱

| # | 支柱 | 說明 |
|---|------|------|
| 1 | **爽劇節奏** | 每一章都是「忍辱→爆發→打臉」，讓人停不下來 |
| 2 | **忍 vs 用的抉擇** | 玩家自己決定什麼時候反擊，越忍越爽 |
| 3 | **用看的就很爽** | 打字機文字 + 像素插畫 + 光效特效，被動也享受 |

### 1.3 目標玩家

> 28 歲上班族，每天通勤 30 分鐘，晚上睡前滑手機。
> 喜歡看大陸爽劇和重生復仇小說，但覺得「看」不夠過癮，想要自己決定什麼時候打臉。
> 不想花時間學操作，但願意看廣告換更多抽卡機會。

### 1.4 商業模式

```
F2P + 廣告 + 可選付費

免費體驗：
├── 完整主線劇情（不鎖章節）
├── 每日免費抽卡
├── 看廣告：重生/回溯/加速累積/解鎖線索/每日禮包
└── 自然回復體力

付費加值（之後版本）：
├── 去廣告包
├── 月卡（每日抽卡石 + 體力）
├── 隱藏劇情線解鎖
└── 限定卡池
```

### 1.5 目標平台

📱 iOS + Android（Godot 4 匯出）

### 1.6 參考遊戲

| 遊戲 | 學習 | 差異化 |
|------|------|--------|
| 叫我官老爺 | 表演式互動（升堂用刑打臉）| 現代商戰 + 像素風 + 深度劇情 |
| 大陸短劇 App | 爽劇節奏 + 廣告變現模式 | 有互動（抽卡+抉擇影響劇情）|
| 修仙文字放置 | 碎片化遊玩 + 章節制 | 非放置，主動推進 + 風險抉擇 |

## 2. 核心玩法

### 2.1 核心循環

```
┌─────────────────────────────────────────────────┐
│                                                   │
│  📖 劇情鋪陳                                     │
│   ↓                                               │
│  😈 壞人使壞 → 囂張值 ↑                          │
│   ↓                                               │
│  😤 主角隱忍 → 抽卡值累積                        │
│   ↓                                               │
│  🔔 懷錶發光 → 抽卡！                            │
│   ↓                                               │
│  ┌──────┴──────┐                                  │
│ 「用！」     「忍！」                              │
│  ↓              ↓                                  │
│ 囂張值 ↓     壞人更囂張                           │
│ 打臉演出     被欺負劇情                           │
│ 小爽 ✅      囂張值繼續 ↑                         │
│  ↓              ↓                                  │
│  └──────┬──────┘                                  │
│         ↓                                          │
│  ⚠️ 囂張值 100% → Game Over → 看廣告重生         │
│  ✅ 囂張值歸零 → 章節完成 → 下一章               │
│                                                   │
└─────────────────────────────────────────────────┘
```

**為什麼不會膩：**
- 每章對手不同，使壞方式不同
- 卡牌隨機，每次打臉方式不同
- 忍的程度不同，劇情分支不同
- 後期章節抽卡次數更多，節奏更緊湊

### 2.2 操作方式

| 操作 | 方式 |
|------|------|
| 推進文字 | 點擊螢幕 / 自動播放 |
| 抽卡 | 點擊懷錶 UI |
| 用卡 / 忍 | 二選一按鈕 |
| 跳過 | 長按快進（已讀劇情）|
| 選擇劇情分支 | 選項按鈕（偶爾出現）|

### 2.3 勝敗條件

| 條件 | 說明 |
|------|------|
| **章節勝利** | 將壞人囂張值降至 0，觸發最終打臉 |
| **章節失敗** | 囂張值達到 100%，壞人勝出 |
| **失敗後** | 看廣告 → 重生到記錄點 |
| **單章時長** | 30-45 分鐘（分 3-5 次遊玩完成）|

### 3.3 各系統摘要

#### 文字演出系統
- **目的**：承載劇情，營造氛圍
- **機制**：JSON 驅動劇情資料 → 打字機逐字顯示 → 角色頭像切換 → 旁白/內心戲不同樣式 → 關鍵台詞放大+震動
- **規格**：→ 見 systems/text-display.md

#### 囂張值系統
- **目的**：製造緊張感，驅動「忍 vs 用」抉擇
- **機制**：UI 進度條 → 壞人事件增加 → 用卡減少 → 100% = Game Over → 0% = 章節完成
- **規格**：→ 見 systems/arrogance-meter.md

#### 抽卡系統
- **目的**：核心驚喜感 + 變現入口
- **機制**：抽卡值滿 → 懷錶 UI 打開 → 轉盤動畫 → 依機率出卡 → N/R/SR/SSR
- **規格**：→ 見 systems/gacha.md

#### 忍/用抉擇系統
- **目的**：核心策略深度，差異化賣點
- **機制**：抽到卡後二選一 → 用=立即打臉 → 忍=卡牌暫存，累積倍率，但囂張值繼續升 → 累積用=更高傷害+隱藏劇情
- **規格**：→ 見 systems/endure-or-strike.md

#### 打臉演出系統
- **目的**：爽感爆發點
- **機制**：依卡牌類型播放不同演出 → 角色抖動 → 大字「啪！」「滾！」→ 螢幕閃白震動 → 囂張值條動畫下降
- **規格**：→ 見 systems/face-slap.md
