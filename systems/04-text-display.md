# 文字演出系統規格

## 基本資訊
- **優先級**：Must（劇情載體）
- **依賴系統**：章節管理
- **被依賴**：囂張值、抽卡系統、CG 插畫

---

## 1. 設計目標

1. 讓玩家沉浸在劇情中
2. 用文字節奏控制情緒起伏
3. 單手操作，通勤可玩

---

## 2. 核心機制

### 文字類型
| 類型 | 顯示方式 | 範例 |
|------|----------|------|
| 對話 | 角色頭像 + 名牌 + 文字框 | 朴在成：「你以為你是誰？」|
| 旁白 | 全寬文字，斜體 | *辦公室的空氣凝結了。* |
| 內心戲 | 主角頭像 + 藍色文字框 | （這傢伙...上輩子就是這張嘴臉）|
| 系統旁白 | 懷錶圖示 + 金色文字 | 〔懷錶微微震動...〕|
| 關鍵台詞 | 放大 + 震動 + 停頓 | **「滾。」** |

### 打字機效果
- 預設速度：每字 40ms
- 關鍵台詞：每字 80ms（慢速強調）
- 玩家點擊：立即顯示完整文字
- 自動播放模式：每段停留 2 秒後自動下一段

### 操作
| 操作 | 效果 |
|------|------|
| 點擊螢幕 | 下一段文字 / 顯示完整 |
| 長按 | 快進（已讀劇情跳過）|
| 左滑 | 回看上一段 |
| 右上角按鈕 | 自動播放 ON/OFF |

```json
{
  "chapter": "ch1",

  "background": "bg_office_daytime",
  "bgm": "bgm_tense_office",
  "segments": [
    {
      "type": "narration",
      "text": "早上九點，姜旻赫準時走進辦公室。一切看起來和上輩子一模一樣。"
    },
    {
      "type": "dialogue",

      "portrait": "zhao_arrogant",
      "text": "喲，姜旻赫，你的報告呢？昨天不是說好今天交嗎？"
    },
    {
      "type": "inner_thought",
      "speaker": "姜旻赫",
      "portrait": "lin_calm",

    },
    {
      "type": "arrogance_event",

      "trigger_text": "朴在成把文件甩到桌上。",
      "animation": "screen_shake_light"

    {
      "type": "key_dialogue",
      "speaker": "朴在成",

    {
      "type": "system_narration",
      "text": "懷錶微微震動，錶蓋泛起淡淡的金光...",
      "trigger": "gacha"

  ]
}
```
