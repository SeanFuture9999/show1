# 風格統一化方案（待執行）

## 問題
PixelLab 每張圖獨立生成，風格會飄移：角色長相、像素密度、色調、線條粗細都不一致。

## 解決方案（優先順序）

### 方案 1：基準圖 + img2img
1. 生成 1 張「黃金基準圖」（Sean 確認風格 OK）
2. 之後所有圖用 img2img API，以基準圖當 style reference
3. PixelLab API endpoint: `/v1/generate-image-pixflux` 支援 `init_image` 參數
4. 預期效果：色調、像素密度、整體氛圍統一

### 方案 2：角色設計表（Character Sheet）
1. 為每個角色生成一張多角度設計表（正面/側面/表情）
2. 用設計表當 reference 生成該角色的所有場景圖
3. 確保同一角色長相一致

### 方案 3：後製統一
1. 所有圖生成後，用統一的色彩校正（調色板限制）
2. 像素密度統一：resize 到同一解析度
3. 描邊粗細統一：後處理加工

### 方案 4：PixelLab Character 功能
- 待確認 API 是否支援 character consistency 功能
- 官方文件需要再查

## 建議執行順序
1. ⭐ 先用方案 1 測試 img2img 一致性效果
2. 效果好 → 建立基準圖庫（每個場景類型 1 張基準）
3. 搭配方案 2 → 角色設計表確保人物一致
4. 最後方案 3 → 後製微調收尾
